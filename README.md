# 📈 Stock Portfolio Tracker

A real-time stock portfolio tracker built with React, Tailwind CSS, and Alpha Vantage API.

## 🔗 Live Demo
[View Live →](https://stock-portfolio-tracker-rosy-psi.vercel.app)

## ✨ Features
- Real-time stock prices via Alpha Vantage API
- Add / remove stocks from your portfolio
- Live P&L calculations per stock
- Portfolio allocation pie chart
- P&L bar chart by stock
- Data persists via localStorage

## 🛠️ Tech Stack
React · Vite · Tailwind CSS · Recharts · Alpha Vantage API · React Router

## 🚀 Run Locally
\`\`\`bash
git clone https://github.com/thupakula13/stock-portfolio-tracker
cd stock-portfolio-tracker
npm install
echo "VITE_ALPHA_VANTAGE_KEY=your_key" > .env
npm run dev
\`\`\`
